Инструкция по подключению OptoVizor к МойСклад

Интеграция OptoVizor позволяет просматривать каталоги поставщиков, обновления ассортимента, остатки и цены, а также использовать интеллектуальный поиск прямо в интерфейсе МойСклад. Подключение выполняется через стандартный механизм приложений МойСклад и завершается в Telegram-боте OptoVizор.

1. Установка приложения

В каталоге решений МойСклад нажмите «Установить». После этого откроется стандартное окно подтверждения доступа. OptoVizor использует только безопасный режим чтения данных (товары, склады, остатки, цены).

2. Подтверждение прав

Нажмите «Разрешить», чтобы активировать приложение. После активации МойСклад передаст безопасный токен в систему OptoVizor. На этом шаге установка в МойСклад завершена.

3. Завершение подключения в Telegram

Чтобы завершить подключение, необходимо выполнить привязку аккаунта МойСклад к вашему аккаунту OptoVizor через Telegram-бота.

В боте откройте раздел «Настройки → Подключить МойСклад» и следуйте инструкции. После успешной привязки вы сможете работать с каталогом поставщиков в OptoVizor с использованием данных вашего МойСклад.

Важно: интеграция начнёт работать только после завершения привязки в Telegram.

4. Работа с каталогом OptoVizor

После привязки вы сможете:

  • просматривать каталоги поставщиков;
  • получать актуальные цены и остатки с разных складов;
  • использовать интеллектуальный поиск по товарным позициям;
  • выбирать склад, по которому требуется показать остатки;
  • работать с каталогом прямо внутри Telegram-бота OptoVizor.

Важная информация

  • Интеграция OptoVizor работает в режиме только чтения и не изменяет данные в МойСклад.
  • Telegram-бот нужен только для привязки аккаунта и работы с каталогом.
  • Интеграция бесплатна для пользователей, имеющих доступ к OptoVizor.
  • Приложение можно удалить в любой момент через раздел «Приложения → Установленные».
