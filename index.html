Инструкция по подключению OptoVizor к МойСклад

Интеграция OptoVizor позволяет просматривать каталог поставщиков, искать товары с помощью ИИ и работать с актуальными ценами прямо в интерфейсе МойСклад. Подключение простое и занимает меньше одной минуты.

1. Установка приложения

Нажмите кнопку «Установить» в каталоге приложений МойСклад. После этого откроется окно с запрашиваемыми правами доступа. OptoVizor использует только безопасный режим чтения и не изменяет ваши данные.

2. Подтверждение прав

Нажмите кнопку «Разрешить». Это свяжет ваш аккаунт МойСклад с платформой OptoVizor. Никакие данные не будут изменены — интеграция работает только в режиме чтения.

3. Раздел OptoVizor в меню МойСклад

После установки в боковом меню МойСклад появится новый пункт «OptoVizor». Перейдите в него, чтобы открыть каталог поставщиков.

4. Работа с каталогом

Вы можете использовать ИИ-поиск по товарам, просматривать цены, наличие и сравнивать предложения поставщиков. Интеграция работает только в режиме чтения и не вносит изменений в ваши данные МойСклад.

Важная информация

  • Telegram-бот НЕ обязателен для пользователей МойСклад. Он используется только поставщиками OptoVizor для подтверждения каталога.
  • Интеграция бесплатна для пользователей с активным доступом к OptoVizor.
  • Приложение не изменяет и не создаёт данные в МойСклад.
  • Вы можете удалять интеграцию в один клик в разделе «Приложения».
